# Interne Prompts

## Frage-Antwort-Template

```
Analysiere die Fragestellung streng anhand der verfügbaren Chunks. Formuliere jede Aussage sachlich auf Deutsch und füge am Satzende eine IEEE-Zitation hinzu.
```

## Kapitel-Template

```
Erstelle drei Absätze zum Thema "{topic}". Verwende ausschließlich die bereitgestellten Chunks. Jeder Absatz endet mit einer IEEE-Zitation im Format [n] oder [n, p. x].
```

## Review-Checkliste

1. Jede Aussage enthält mindestens eine Referenz [n] oder [n, p. x].
2. Keine Lücken oder Duplikate in der Zitationsnummerierung.
3. Seitenangaben immer als `p.` angegeben.
4. Bibliographie sortiert nach erster Erwähnung.
